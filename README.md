# programming-FAQ

新手的最大问题是不知道该问什么问题，所以如果你想学习编程，那么首先：

1. 找到自己学习编程的目的。
2. 学会翻墙急及善用 Google。如果不会翻，则用 Bing，总之远离百度。
3. 按照推荐的一份教程慢慢开始，遇到不懂的再问（注册一个 GitHub 账号）。
